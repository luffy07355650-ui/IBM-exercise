# IBM Data Science Capstone Project

This repository contains the completed materials for the IBM Data Science Capstone.

## Contents
- Jupyter Notebooks (.ipynb) for data collection, wrangling, EDA, SQL analysis, visualization, and machine learning
- A Plotly Dash dashboard implemented as a Python script (.py)
- Final presentation submitted in PDF format on Coursera

## Notes
Most analytical tasks are implemented in Jupyter Notebooks.
The interactive dashboard is implemented as a Python script following the course instructions.
